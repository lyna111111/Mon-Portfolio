│ 1️⃣  Créer un compte sur https://www.emailjs.com/                        │
│ 2️⃣  Configurer Gmail comme service email                                │
│ 3️⃣  Créer un template d'email                                           │
│ 4️⃣  Récupérer vos 3 clés :                                              │
│     - Public Key                                                         │
│     - Service ID                                                         │
│     - Template ID                                                        │
│ 5️⃣  Mettre à jour Contact.js avec vos clés OU créer un fichier .env     │
│ 6️⃣  Tester le formulaire !                                              │

┌──────────────────────────────────────────────────────────────────────────┐
│ OÙ METTRE VOS CLÉS ?                                                     │
├──────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│ Option A : Directement dans le code (src/components/Contact.js)         │
│ ─────────────────────────────────────────────────────────────────────    │
│   Ligne 44 : emailjs.init("VOTRE_PUBLIC_KEY");                          │
│   Ligne 72 : 'VOTRE_SERVICE_ID'                                          │
│   Ligne 73 : 'VOTRE_TEMPLATE_ID'                                         │
│                                                                          │
│ Option B : Variables d'environnement (Recommandé)                       │
│ ─────────────────────────────────────────────────────────────────────    │
│   1. Créez un fichier .env à la racine                                  │
│   2. Ajoutez :                                                           │
│      REACT_APP_EMAILJS_PUBLIC_KEY=votre_clé                              │
│      REACT_APP_EMAILJS_SERVICE_ID=votre_id                               │
│      REACT_APP_EMAILJS_TEMPLATE_ID=votre_id                              │
│   3. Redémarrez npm start                                               │
│                                                                          │
