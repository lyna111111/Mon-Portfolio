                    📝 METTRE À JOUR LE CODE
═══════════════════════════════════════════════════════════════════════════

Maintenant que vous avez vos 3 clés :

1️⃣  Ouvrez Visual Studio Code

2️⃣  Ouvrez le fichier : src/components/Contact.js

3️⃣  Modifiez 3 lignes :

    Ligne 44 : Remplacez "YOUR_PUBLIC_KEY" par votre Public Key
    Ligne 72 : Remplacez 'YOUR_SERVICE_ID' par votre Service ID
    Ligne 73 : Remplacez 'YOUR_TEMPLATE_ID' par votre Template ID

4️⃣  Sauvegardez (Ctrl + S)

5️⃣  Testez sur http://localhost:3000
