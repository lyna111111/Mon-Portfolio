🎯 LES 3 CLÉS À RÉCUPÉRER
─────────────────────────

1️⃣  PUBLIC KEY
    Où ? EmailJS > Account > General > API Keys
    Exemple : AbCdEfGhIj1234567890

2️⃣  SERVICE ID
    Où ? EmailJS > Email Services > Votre service Gmail
    Exemple : service_abc1234

3️⃣  TEMPLATE ID
    Où ? EmailJS > Email Templates > Votre template
    Exemple : template_xyz7890

📍 OÙ METTRE LES CLÉS ?
───────────────────────

Fichier : src/components/Contact.js

Ligne 44 : Remplacez "YOUR_PUBLIC_KEY" par votre Public Key
Ligne 72 : Remplacez 'YOUR_SERVICE_ID' par votre Service ID
Ligne 73 : Remplacez 'YOUR_TEMPLATE_ID' par votre Template ID

💡 APRÈS AVOIR MIS LES CLÉS
───────────────────────────

1. Sauvegardez le fichier (Ctrl + S)
2. Allez sur http://localhost:3000
3. Testez le formulaire de contact
4. Vérifiez votre email lyna.can20@gmail.com

🆘 BESOIN D'AIDE ?
──────────────────

Si vous voyez "Erreur lors de l'envoi" :

1. Vérifiez que les 3 clés sont correctes
2. Vérifiez qu'il n'y a pas d'espaces avant/après les clés
3. Ouvrez la console du navigateur (F12) pour voir l'erreur
4. Consultez la section DÉPANNAGE dans CONFIGURATION_EMAILJS_DETAILLEE.md
